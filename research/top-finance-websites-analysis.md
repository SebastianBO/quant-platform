### Programmatic SEO Strategy

**Scale:** Tens of thousands of filterable combinations

**URL Structure:**
```
/screener.ashx                                    # Base screener
/screener.ashx?v=111&f=cap_large,exch_nyse       # Filtered view
/quote.ashx?t={TICKER}                           # Stock quote
/map.ashx?t=sec                                  # Sector map
/map.ashx?t=geo                                  # Geographic map
/futures.ashx                                    # Futures overview
/forex.ashx                                      # Forex pairs
/groups.ashx?g=sector&v=110&o=-volume           # Group analysis
```

**URL Parameter System:**
- `v=` View type (table layout)
- `f=` Filters (comma-separated codes)
- `o=` Order/sort column
- `t=` Ticker or map type
- `s=` Signal/preset screen

**Filter Code Examples:**
- `cap_largeover` - Large cap+
- `exch_nasd` - NASDAQ
- `fa_div_o5` - Dividend > 5%
- `ta_sma20_pa` - Price above SMA20
- `idx_sp500` - S&P 500 members

**SEO Advantage:**
